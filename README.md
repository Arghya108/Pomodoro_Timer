# Pomodoro_Timer
<img width="602" height="821" alt="image" src="https://github.com/user-attachments/assets/d7658fb8-88cc-4c7c-95f9-ff15b997aa17" />
A clean, dark-themed desktop Pomodoro Timer built with Python and Tkinter. Supports 25/5 work-break cycles with real-time countdown, pause/resume, auto session switching, customizable durations, session tracking, and sound alerts — all using standard Python libraries only.
